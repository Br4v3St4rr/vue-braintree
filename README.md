## Installation
``` js
yarn add vue-braintree
```
``` js
import VueBraintree from 'vue-braintree'
Vue.use(VueBraintree)
```

## Build Setup

``` bash
# install dependencies
yarn

# build for production with minification
npm run build

# run unit tests
npm test

# run unit tests with hot reload
npm run watch
```
